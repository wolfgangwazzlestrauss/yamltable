Welcome to YamlTable's documentation
====================================

YamlTable is a command line utility for working with YAML files, that are organized as a list of similar
dictionaries.

Commands:

* ``list``: takes a key name and lists each dictionary key value
* ``sort``: takes a key name and sorts the dictionaries by the key values.
